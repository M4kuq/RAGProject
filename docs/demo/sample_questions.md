# Sample Questions

seed 文書と evaluation fixture に合わせた質問である。

## Good Context

1. What vector database is used by Phase1?
2. How does Phase1 keep CI deterministic?
3. What retrieval trace capability does the seed document mention?
4. Where are confidence labels stored?
5. Which MCP transport is used in Phase1?
6. What can an admin do with documents?
7. What is the default evaluation dataset?
8. Why does Phase1 use Docker Compose?

## No Context

1. What is the office Wi-Fi password?
2. What AWS account is used for production?
3. Who approved the private customer contract?

No Context の質問は、文書にない内容を断定しないことを確認するために使う。
